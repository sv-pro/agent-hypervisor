How to use this demo

  1. Click any of the scenarios above (they will open in a new tab).
  2. Open your Chrome Side Panel and navigate to the Browser Agent Hypervisor Demo.
  3. Ensure you are in the Governed tab.
  4. Experiment with the buttons: Summarize (read-only actions are allowed), and Save note (which will ask or deny based on the Deterministic Rules).
  5. Check the Trace tab in the side panel to see exactly which rule (e.g., RULE-SM-HIDDEN) fired for each decision!
