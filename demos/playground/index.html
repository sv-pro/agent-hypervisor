Interactive Demo

Capability Rendering
Playground

Step through taint propagation, ontology projection, and policy evaluation live. Feed arbitrary inputs. Watch which actions survive rendering and which disappear.

Requires local server — run instructions below

The playground is a full interactive demo with a live policy engine backend. It runs locally — type any agent input, choose a manifest and tool, and see exactly how the hypervisor evaluates it: taint classification, ontology validation, provenance check, and three-way verdict.

Taint Flow
Input provenance

Every input is labeled with its trust class — user-declared, external document, derived, or system. Watch taint propagate through the pipeline.

Ontology
Manifest projection

Select a manifest. The available action space narrows to what the manifest permits. Out-of-manifest calls are denied before execution.

Policy
Three-way verdict

Each action receives allow, deny, or ask. Policy rules are readable — you can see exactly which rule triggered and why.

Scenarios
Preloaded attacks

Pre-built scenarios demonstrate prompt injection, exfiltration attempts, and privilege escalation — and how each is handled structurally.

Run locally
bash
# install dependencies
cd playground
npm install
 
# start server + open in browser
npm run dev
→ http://localhost:5173
 
# alternatively: run the static demo
npm run server # starts Fastify backend on :3000
open http://localhost:3000
