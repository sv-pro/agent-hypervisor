# Research and Explorations

This "messy" space is an index cataloging raw explorations, older iterations, and running experiments located in the repository's `_research` directory. It stands separate from the clean canonical concepts. 

Documents and code here may eventually be synthesized and promoted to the canonical wiki.

## Directory Structure Overview

- **[`agentdojo-bench/`](../../_research/agentdojo-bench/) & [`benchmarks/`](../../_research/benchmarks/)**: Explorations into benchmarking the Agent Hypervisor architecture. References external test constraints against deterministic outcome validations.
- **[`archive/`](../../_research/archive/)**: Deprecated exploratory files.
- **`demos/` & `examples/`**: Extensive legacy concept implementations predicting the [Four-Layer Architecture](../concepts/architecture.md).
  - Highlights include [`examples/basic/`](../../_research/examples/basic/), [`examples/claude_like_runtime/`](../../_research/examples/claude_like_runtime/), [`examples/compiler/`](../../_research/examples/compiler/), and the [`runtime/provenance_firewall/`](../../_research/examples/runtime/provenance_firewall/) mechanisms that established the [Taint Rules](../concepts/trust-and-taint.md) for the core framework.
- **[`docs/`](../../_research/docs/)**: Legacy doc assets.
- **`experiments/`**: Working prototype scripts verifying boundary isolation functionality.
  - [`dspy_workflow_capability/`](../../_research/experiments/dspy_workflow_capability/)
  - [`dspy_world_proposal/`](../../_research/experiments/dspy_world_proposal/)
- **[`lab/`](../../_research/lab/)**: Scratch space for one-off prototyping (e.g., [`pipeline_walkthrough.ipynb`](../../_research/lab/pipeline_walkthrough.ipynb)).
- **[`reports/`](../../_research/reports/) & [`traces/`](../../_research/traces/)**: Execution output capturing exactly how taint paths flow and operate inside isolated environments, acting as literal representations of [Manifest Resolution Law](../concepts/manifest-resolution.md).
- **[`tests/`](../../_research/tests/)**: Legacy testing specifications for older configurations.
